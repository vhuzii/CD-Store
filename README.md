# CD-Store

CD Marketplace — Online Disk Shop

Цей проект є веб-додатком маркетплейсу для продажу музичних дисків.
Користувачі можуть реєструватися, додавати власні диски на продаж, переглядати товари інших користувачів, спілкуватися між собою та оформляти покупки.

Проект створений як навчальна робота для демонстрації роботи з веб-технологіями, UI/UX та backend-логікою.
